====================================
  抖音直播间自动点赞助手
====================================

快速开始:
1. 双击 DouYin_AutoClik.exe 启动程序
2. 扫码登录（推荐）
3. 输入直播间链接
4. 配置参数后开始任务

详细文档请访问: https://github.com/jaxwang1991/DouYin_AutoClik

功能特性:
- 自动点赞（拟人化点击）
- AI 智能评论（需配置 API）
- 音频转录（DashScope）
- 循环模式
- 验证码自动检测

注意事项:
- 首次使用请扫码登录
- 合理设置点击频率和休息时间
- AI 评论功能需要配置 DashScope API 密钥
