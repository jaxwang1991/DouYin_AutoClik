# 抖音直播间自动点赞助手 (DouYin Live Auto Liker)

基于 Python + Playwright 实现的抖音直播间自动点赞工具。模拟真人点击操作，支持图形化界面配置，内置多种防风控策略，稳定高效。

## ✨ 核心功能

*   **图形化界面 (GUI)**: 直观的操作面板，轻松配置各项参数。
*   **智能循环模式 (防风控)**:
    *   默认**开启**。支持设置“工作 X 分钟，休息 Y 分钟”的循环机制。
    *   **实时倒计时**: 休息状态下，底部状态栏实时显示剩余休息时间（XX分XX秒）。
*   **拟人化点击**:
    *   **随机间隔**: 支持设置快速连点和慢速停顿的随机时间范围，模拟真实用户节奏。
    *   **随机位置**: 点击位置在视频区域内随机分布，避免固定坐标被检测。
*   **全自动异常处理**:
    *   **验证码检测**: 自动识别滑块/拼图等验证码并暂停任务，弹出提示等待人工处理，验证完成后自动恢复。
    *   **手速冷却**: 检测到“手速太快”提示时，自动进入冷却倒计时（默认 3 分钟），结束后自动恢复。
    *   **状态监测**: 自动检测直播是否结束或视频流断开，智能停止任务。
*   **灵活运行模式**:
    *   **可视化/静默运行**: 支持选择是否显示浏览器窗口。默认**显示浏览器**，方便观察运行状态；也可勾选“后台静默运行”以隐藏窗口。
*   **账号管理**:
    *   **扫码登录**: 内置扫码登录功能，自动保存登录状态 (`state.json`)，一次登录长期有效。
    *   **游客模式**: 未登录状态下也可以游客身份运行（部分功能可能受限）。

## 🚀 快速开始

### 1. 环境准备
确保电脑已安装 Python。
双击运行项目目录下的 **`setup.bat`**：
- 自动安装所需的 Python 依赖库 (playwright 等)。
- 自动安装浏览器驱动。

### 2. 启动软件
双击运行 **`run_gui.bat`** 即可打开图形化操作界面。

### 3. 使用步骤
1.  **账号登录** (推荐): 点击界面上的“扫码登录”按钮，使用抖音 App 扫码。登录成功后关闭浏览器即可。
2.  **输入链接**: 在“直播间链接”输入框中粘贴目标直播间的分享链接或网址。
3.  **调整配置**:
    - **循环模式**: 默认已开启（工作 10 分钟，休息 5 分钟），可根据需要调整。
    - **运行模式**: 默认会显示浏览器窗口，如需后台运行请勾选“后台静默运行”。
4.  **开始点赞**: 点击“开始任务”按钮。

## 🛠️ 注意事项

*   **验证码处理**: 当程序检测到验证码时，会暂停点击并弹窗提示。请切换到弹出的浏览器窗口，手动完成验证码（如滑动拼图），验证成功后程序会自动检测并恢复运行。
*   **适度使用**: 虽然软件有防风控机制，但建议合理设置点击频率和休息时间，避免单次运行时间过长（建议 2-3 小时重启或休息）。
*   **日志查看**: 界面下方的“运行日志”会实时显示当前操作、状态变化及错误信息。

## 📂 文件说明
- `gui.py`: 图形化界面主程序。
- `liker.py`: 核心点赞逻辑与浏览器控制。
- `auth.py`: 扫码登录脚本。
- `setup.bat`: 环境安装脚本。
- `run_gui.bat`: 软件启动脚本。
